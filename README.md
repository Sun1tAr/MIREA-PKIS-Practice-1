
# MIREA-PKIS-Practice-1

Консольное приложение для работы с текстовыми файлами. Практическая работа №1 по дисциплине "Программирование корпоративных информационных систем" (ПКИС).

## 🚀 Функциональность

Приложение позволяет выполнять следующие операции с текстовыми файлами:

| № | Операция | Описание |
|---|----------|----------|
| 0 | Завершить работу | Выход из программы |
| 1 | Ввести путь к файлу | Указание пути к текстовому файлу для обработки |
| 2 | Вывести текст в консоль | Отображение содержимого файла |
| 3 | Найти подстроку в файле | Поиск количества вхождений подстроки |
| 4 | Посчитать количество слов в файле | Подсчёт слов в текстовом файле |

## 🏗️ Архитектура проекта

```
└── my.learn
├── console          # Консольное взаимодействие
│   ├── AppExecutor   # Исполнитель команд пользователя
│   └── ConsoleSender # Отправка сообщений в консоль
├── constants         # Константы приложения
├── model            # Модели данных
│   ├── message       # Система сообщений
│   │   ├── Message (интерфейс)
│   │   ├── DescriptionMessage
│   │   ├── OperationMessage
│   │   └── ResultMessage
│   └── Operation     # Enum операций
└── service          # Бизнес-логика
├── FileService   # Работа с файлами
└── OperationsService # Обработка команд
```

## 💡 Особенности реализации

### 🎯 Message System
Проект использует полиморфную систему сообщений:
- **Message** - интерфейс для всех типов сообщений
- **OperationMessage** - отображение списка операций
- **DescriptionMessage** - запрос дополнительных данных
- **ResultMessage** - результат выполнения операции

### 🔄 Управление файлами
- Валидация пути к файлу
- Обработка ошибок при чтении
- Автоматическая инвалидация при проблемах с файлом

### 🛡️ Обработка ошибок
- Проверка корректности ввода
- Обработка исключений на верхнем уровне
- Информативные сообщения об ошибках

## 🛠️ Технологии

- **Java 17**
- **Maven** - сборка проекта
- **Lombok** - уменьшение шаблонного кода
- **JUnit 5 / Mockito** - тестирование (планируется)

## 📦 Сборка и запуск

### Предварительные требования
- JDK 17 или выше
- Maven 3.6+

### Сборка проекта
```bash
mvn clean package
```

### Запуск приложения
```bash
java -jar target/MIREA.PKIS.Practice-1-1.0.jar
```

Или через Maven:
```bash
mvn exec:java -Dexec.mainClass="my.learn.Main"
```

## 🎮 Использование

1. Запустите приложение
2. Введите путь к файлу (операция 1)
   ```
   Пример: C:\Users\user\Desktop\test.txt
   ```
3. Выберите нужную операцию:
    - **2** - просмотр содержимого
    - **3** - поиск подстроки
    - **4** - подсчёт слов
4. Для завершения работы выберите операцию 0

### Пример работы
```
-----------------------------------------------
Текущий файл: C:\Users\user\Desktop\test.txt
-----------------------------------------------
0 - Завершить работу
1 - Ввести путь к файлу
2 - Вывести текст в консоль
3 - Найти подстроку в файле
4 - Посчитать количество слов в файле
-----------------------------------------------
> 2
Содержимое файла:
Hello world! This is a test file.
```

## 🧪 Тестирование

Тестами покрывается только основной сервис приложения (FileService).

Тестирование верхнеуровневых компонентов (консольное взаимодействие, основной цикл) не планируется, так как они являются по сути интерфейсами и их тестирование нецелесообразно.
